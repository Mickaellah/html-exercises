All elements

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The abbreviation element (<abbr>) is an element that is useful in HTML.

The (<abbr>) tag defines an abbreviation or an acronym, like "HTML", "Mr", "Dec", etc. An abbreviation and an acronym are both shortened versions of something else.

<article>

The article element (<article>) should make sens on its own.

The (<article>) specifies independent, self-contained content.

<button>

The button element (<button>) defines a clickable button.

Inside of the (<button>) element you can put content, like text or images.

<body>

The body element (<body>) contains all the contents of an HTML document such as hyperlinks.

The (<body>) defines the document's body.

<br>

The (<br>) is useful for writing where the division of lines is significant.

The (<br>) is an inline element used to create a single line break in a document.

<code>

The (<code>) tag defines a fragment of comuter code in the HTML document.

The (<code>) element dislpays it contents style in a fashion intended to indicate that the text is a short.

<doctype>

The (<doctype>) is a document type html.

The (<doctype>) element is an instraction to the web browser about what version HTML the page is written in.

<dl>

The (<dl>) element represents a description list.

The (<dl>) is used for declaring a definition list.

<dt>

The HTML (<dt>) element specifies a term in a description or definition list.

The dt element represents the term, or name, part of a term-description group in a description list (dl element).

<dd>

The <dd> is an element we use to describe a term/name in a description list.

The dd element represents the description, definition, or value, part of a term-description group in a description list (dl element)

<div>

The (<div>) element has no special meaning at all.

The DIV element represents its children.

<footer>

The (<footer>) represents the "footer" of a document or section of a document.

The "footer" element contains information about the author of the section.

<html>

The HTML element defines a whole document.

The HTML element represents the root of an HTML document.

<head>

The "head" element is used to write a title of a document.

The "head" element is a container for data about data and is placed between <html> tag and <body> tag.

<header>

The (<header>) element is used to define the header for a document or a section.

The header element contains information related to the title and heading of the related content.

<h1>

H1 is the highest heading level.

The h1 element is used to indicate the most important heading on the page.

<h2>

H2 is the second highest heading level.

The HTML h2 element represents a level 2 heading in an HTML document.

<h3>

H3 is the medium heading level.

The HTML h3 element represents a level 3 heading in an HTML document.

<h4>

H4 is the medium heading level after h3.

The HTML h4 element represents a level 4 heading in an HTML document.

<h5>

H5 is the second lowest heading level.

The HTML h5 element represents a level 5 heading in an HTML document.

<h6>

H6 is the lowest heading level.

The HTML h6 element represents a level 6 heading in an HTML document.

<img>

The (<img>) element defines an image in an HTML page.

The IMG element creates a holding space for the referenced image

<iframe>

The iframe element represents a nested browsing context.

The Iframe element defines a rectangular region within the document in which the browser can display a separate document, including scrollbars and borders.

<link>

The (<link>) element contains attributes only.

The link element allows users to link their document to other resources.

<li>

The (<li>) element is used to make a list in html.

The (<li>) element is a child of parent element like (<ol>) or (<ul>).

<meta>

The (<main>) element represents the dominant content of the (<body>) of a document.

Inside of the (<main>) element, there could be lots of other elements.

<nav>

The (<nav>) element is intended only for major block of navigation links.

The (<nav>) element is a section containing links to other documents.

<ol>

The (<ol>) is useful for creating lists that are either numbered or alphabetical.

The (<ol>) element is used for making an ordered list in a document.

<p>

The (<p>) element represents a paragraph.

The (<p>) elements are only allowed to contain inline elements.

<style>

The (<style>) element is used to define style information for an HTML document.

Inside of the (<style>) element we specify how HTML elements should render in a browser.

<summary>

The (<summary>) element is useful to make a summary of a document.

The (<summary>) element specifies a summary or caption for a (<details) element's disclosure box.

<script>

The (<script>) element is used to define a client-side script.

The script element allows users to include dynamic script and data blocks in their documents.

<section>

The (<section>) element is useful for making division in a document.

The (<section>) element defines sections in a document.

<th>

The (<th>) element is useful when you create a table in html.

The (<th>) element represents a header cell in a table.

<tr>

The (<th>) element is useful when you create a table in html.

The (<th>) element represents a row of cells in a table.

<td>

The (<th>) element is useful when you create a table in html.

The (<th>) element represents a data cell in a table.

<thead>

The (<thead>) element is used to group header content in an html table.

The (<thead>) element represents the block of rows that consist of the column labels.

<tbody>

The (<tbody>) element is used to group the body content in html table.

The (<tbody>) element represents the block rows that consist of a body of data.

<tfoot>

The (<tfoot>) element is used to group footer content in html table.

The (<tfoot>) element represents the block rows that consist of the column summaries.

<ul>

The (<ul>) element is useful for making an unordered list in html.

The (<ul>) element represents a list of items, where the order of the items is not important.

<del>

The (<del>) tag defines text that has been deleted from a document.

The (<del>) element represents a range of text that has been deleted from a document.

<dfn>

The (<dfn>) tag represents the defining instance of a term in HTML.

The (<dfn>) element is used to identify the defining instance of a term in an HTML document.

<em>

The (<em>) element is used to emphasize or stress a word or a phrase.

The (<em>) element renders as emphasized text.

<ins>

The element (<ins>) is used to add a word or group of words to the middle of a document or piece of writing.

The element (<ins>) represents a range of text that has been added to a document.

<kbd>

The element (<kbd>) is used to present a keyboard shortcut.

The element (<kbd>) defines text as user input from a keyboard such as the Enter or Ctrl keys.

<mark>

The element (<mark>) is a type of phrasing content element.

The element (<mark>) represents a run of text in one document marked or highlighted for reference purposes.

<pre>

The element (<pre>) is used to define preformatted text.

The element (<pre>) represents preformatted text which is to be presented exactly as written in the HTML file.

<q>

The element (<q>) is used to wrap a word or a sentence in quotes.

The element (<q>) indicates that the enclosed text is a short inline quotation.

<s>

The element (<s>) is used to put a line through text in html document.

The element (<s>) represents text that is no longer accurate or relevant.

<small>

The element (<small>) represents side-comments and small print.

The element (<small>) only includes the global attributes.

<span>

The element (<span>) is used to group inline elements in a document.

The element (<span>) is a generic inline container for phrasing content.

<strong>

The (<strong>) element is used to separate the text from the rest of the content.

The (<strong>) element gives text a strong emphasis.

<sub>

The (<sub>) element is used to add a subscribpt text to the html document.

The (<sub>) element defines the subscipt text.

<sup>

The (<sup>) element defines superscript text.

The (<sup>) element specifies inline text which is to be displayed as superscript for solely typographical reasons

<time>

The (<time>) element represents a specific period in time.

The (<time>) element is for presenting dates and times in a machine readable format.

<var>

The (<var>) element is presented using an italicized version of the current typeface.

The (<var>) element represents the name of a variable in a mathematical expression or a programming context.
