All elements

<a>

The anchor element (<a>) is one of the fundamental elements in HTML.

It allows a user to click on the link text it provides, and navigates to that link's location.

<abbr>

The abbreviation element (<abbr>) is an element that is useful in HTML.

The (<abbr>) tag defines an abbreviation or an acronym, like "HTML", "Mr", "Dec", etc. An abbreviation and an acronym are both shortened versions of something else.

<article>

The article element (<article>) should make sens on its own.

The (<article>) specifies independent, self-contained content.

<button>

The button element (<button>) defines a clickable button.

Inside of the (<button>) element you can put content, like text or images.

<body>

The body element (<body>) contains all the contents of an HTML document such as hyperlinks.

The (<body>) defines the document's body.

<br>

The (<br>) is useful for writing where the division of lines is significant.

The (<br>) is an inline element used to create a single line break in a document.

<code>

The (<code>) tag defines a fragment of comuter code in the HTML document.

The (<code>) element dislpays it contents style in a fashion intended to indicate that the text is a short.

<doctype>

The (<doctype>) is a document type html.

The (<doctype>) element is an instraction to the web browser about what version HTML the page is written in.

<dl>

The (<dl>) element represents a description list.

The (<dl>) is used for declaring a definition list.

<dt>

The HTML

element specifies a term in a description or definition list.

The dt element represents the term, or name, part of a term-description group in a description list (dl element).

<dd>

The <dd> is an element we use to describe a term/name in a description list.

The dd element represents the description, definition, or value, part of a term-description group in a description list (dl element)

<div>

The (<div>) element has no special meaning at all.

The DIV element represents its children.

<footer>

The (<footer>) represents the "footer" of a document or section of a document.

The "footer" element contains information about the author of the section.

<html>

The HTML element defines a whole document.

The HTML element represents the root of an HTML document.

<head>

The "head" element is used to write a title of a document.

The "head" element is a container for data about data and is placed between <html> tag and <body> tag.

<header>

The (<header>) element in HTML is used to define the header for a document or a section.

The header element contains information related to the title and heading of the related content.

<h1>

H1 is the highest heading level.

The h1 element is used to indicate the most important heading on the page.

<h2>

H2 is the second highest heading level.

The HTML h2 element represents a level 2 heading in an HTML document.

<h3>

H3 is the medium heading level.

The HTML h3 element represents a level 3 heading in an HTML document.

<h4>

H4 is the medium heading level after h3.

The HTML h4 element represents a level 4 heading in an HTML document.

<h5>

H5 is the second lowest heading level.

The HTML h5 element represents a level 5 heading in an HTML document.

<h6>

H6 is the lowest heading level.

The HTML h6 element represents a level 6 heading in an HTML document.

<img>

The (<img>) element defines an image in an HTML page.

The IMG element creates a holding space for the referenced image

<iframe>

The iframe element represents a nested browsing context.

The Iframe element defines a rectangular region within the document in which the browser can display a separate document, including scrollbars and borders.
